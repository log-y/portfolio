What I've Been Working On

Undergraduate Researcher at Tufts University

Built an ecosystem for developing and distributing audio transcription plugins. (Fall 2024)

Tech used:

GailBot is an audio transcription tool that accounts for the paralinguistic features of audio (like voice intonation, prosody, etc). The API software was an award-winning breakthrough in human interaction, winning the 2023 Sage Concepts Grant Award ($20,000) and being published in a prestigious journal, Dialogue and Discourse. My team and I were building an ecosystem around GailBot, allowing other developers to build and share their own machine learning plugins. We built a new website for uploads and downloads of these plugins using React and AWS. We were also working on a Python package (published to PyPI) where people can access developer tools to continue building their plugins.

Summer Research Assistant at the University of Houston

Used a number of deep learning methods to analyze processes in fluid dynamics. (Summer 2024)

Tech used:

This summer, I joined the Petroleum Engineering Department at the University of Houston to explore safer and more efficient ways of drilling for petroleum. Drilling success depends on the qualities of the rock that you're drilling through. The optimal rock is permeable, porous, even-grained, and naturally fractured. Deviations from that ideal will make drilling more inefficient and dangerous. With my advisor, endowed professor Qin Guan, I coded a project that focused on evaluating the drillability of a shale formation using CT scans of shale samples. I developed a deep learning pipeline that applied methods like convolutional neural networks, feature extraction, and regression to analyze the images and predict key rock properties. Essentially, my program was a big function that took in a 100x100 preprocessed CT scan of shale and returned a .csv file containing a number of useful metrics about the sample.
